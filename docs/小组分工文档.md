## 系统处理流程与模块驱动关系

```
用户上传文件
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│ ① 页面渲染（李锦实）                                    │
│    PDF/图片 → PNG 页面，质量评估                        │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ ② 版面分析（朱珊）                       │
│    版面检测 + 规则清洗：识别标题/正文/图片/表格/页眉页脚 │
│    恢复阅读顺序，标记噪音区域                            │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ ③ OCR 识别（朱珊）                      │
│    PaddleOCR + 百度云 OCR 双引擎，文字识别与置信度评估   │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ ④ 文本清洗（关诗璇）                                    │
│    段落合并、噪声过滤、跨页拼接                          │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ ⑤ 文档结构构建（关诗璇）                                │
│    将清洗后的 block 按阅读顺序组装为章节结构             │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ ⑥ 多格式导出（关诗璇）                                  │
│    EPUB/DOCX/Markdown/HTML 四种格式导出                  │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
                    ┌──────────┐
                    │ 下载文件  │
                    └──────────┘
```

**各模块的上游/下游依赖关系：**

- **王凯燕**（数据库设计）→ 为所有模块提供表结构，是系统的基础设施
- **李锦实**（渲染）→ 产出页面图片 → 交给**朱珊**做版面分析
- **朱珊**（版面分析 + OCR）→ 产出识别框和文字 → 交给**关诗璇**清洗
- **关诗璇**（清洗 + 构建 + 导出）→ 产出最终文档 → 前端展示
- **张婧涵**（翻译功能和AI接口设计）→ 在编辑环节辅助用户翻译和整理
- **朱珊**（前端）→ 用户看到的界面，对接后端 API
