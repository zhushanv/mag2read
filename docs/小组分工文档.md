# Mag2Read 小组分工文档

## 项目概述

Mag2Read 是一个杂志扫描件转电子书的 Web 系统，支持用户上传 PDF/图片，自动完成版面分析、OCR 识别、文本清洗、结构构建，最终导出 EPUB、DOCX、Markdown、HTML 等多种格式。后端基于 FastAPI，前端基于 React + Vite，数据库使用 MySQL，任务调度使用 Celery + Redis。

## 二、技术经理 —— 陈某某（技术方案 + 核心模块开发，30% 工作量的核心代码）

### 版面分析模块
- **版面检测**：`backend/app/modules/layout_detect.py` — 基于 PaddleX 的版面元素检测
- **版面规则清洗**：`backend/app/modules/layout_refine.py` — 检测框的后处理规则，噪声过滤、阅读顺序恢复
- **PaddleX 调用封装**：`backend/app/modules/baidu_paddle_vl.py` — 百度 PaddleX / 百度云 VL API 的调用封装

### OCR 模块
- **OCR 识别**：`backend/app/modules/ocr.py` — PaddleOCR + 百度云 OCR 双引擎实现
- **OCR 引擎策略**：选择、回退、置信度评判

### 基础架构
- **任务数据模型**：`backend/app/models/task.py` — 所有数据库表模型
- **配置管理**：`backend/app/core/config.py` — 环境配置管理
- **数据库连接**：`backend/app/core/database.py`
