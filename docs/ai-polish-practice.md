# AI 一键整理 —— 实践文稿

## 📌 功能概述

「AI 一键整理」是 Mag2Read 提供的一项智能阅读辅助功能。它的定位是：**不改变原文内容，不修改原始清洗稿，而是在阅读界面右侧独立生成一份经过翻译和结构重整的「辅助阅读文稿」**，帮助用户更快地理解外文杂志、期刊内容。

## 🎯 解决的问题

| 痛点 | 解决方式 |
|------|----------|
| 扫描件 PDF 中大量英文段落阅读慢 | 自动检测非中文段落 → 机器翻译为中文 |
| 英文标题、图注切换理解费劲 | 标题/图注级联翻译，保留原文对照 |
| 原文结构松散、无层次感 | 按标题层级（H1→H2→H3→正文）重组 |
| 页眉页脚、页码等噪音干扰阅读 | 自动过滤，只保留正文内容 |
| 读完英文文章后想做中文笔记 | 一键输出整理文稿，可直接复制使用 |

## 🔧 技术实现

### 流程

```
前端点击「一键整理」
        ↓
后端读取当前清洗稿（edited_document.json）
        ↓
提取所有非噪音文本块（按阅读顺序）
        ↓
逐块检测是否以中文为主
        ↓
非中文段落 → 批量调用翻译 API
        ↓
按标题层级重组为 Markdown 格式
        ↓
返回整理文稿 + 统计信息 → 前端展示
```

### 翻译服务选择

本功能使用阿里云机器翻译（通用版）作为首选翻译引擎，备选百度翻译通用版。

| 特性 | 阿里云翻译 | 百度翻译 |
|------|-----------|---------|
| 免费额度 | 100万字符/月 | 200万字符/月 |
| 超出单价 | 50元/百万字符 | 49元/百万字符 |
| 适合场景 | 辅助阅读，按需使用 | 大文档批量翻译 |
| 国内访问 | 低延迟，稳定 | 低延迟，稳定 |

> **建议**：对于普通学生用户，每月 100~200 万字符的免费额度完全足够。一篇 10 页的英文杂志文章，正文翻译大约消耗 3~5 万字符。

### 语言检测

采用基于 Unicode 范围的启发式方法判断文本语言：

```
中文字符占比 = 中文字符数 / 去空格后总字符数
     > 50% → 视作中文段落，跳过翻译
     ≤ 50% → 视作非中文段落，进行翻译
     < 3 字符 → 过短，跳过
```

这种方法不需要额外部署语言检测模型，轻量快速，对客户端无负担。

## 📖 使用场景示例

### 场景一：阅读英文杂志文章

用户上传了一篇《The Economist》的扫描件 PDF，处理完成后：

1. 在左侧阅读稿中看到英文原文
2. 每个非中文段落右下角自动出现「🌐 翻译」按钮
3. 鼠标悬停时按钮高亮，点击后下方 inline 展开灰色底色的中文翻译
4. 翻译结果可随时收起，不干扰原文阅读

### 场景二：需要全文中英对照参考

1. 打开右侧 AI 面板（悬浮按钮）
2. 点击「一键整理」按钮
3. 等待数秒（视段落数量而定）
4. 看到一份结构完整的 Markdown 文稿，每个非中文段落下方附有中文翻译
5. 可直接复制到笔记软件中

### 场景三：需要中文笔记/摘要

1. 执行一键整理后
2. 文稿中保留了原文标题层级
3. 非中文正文段落后方附有斜体翻译
4. 可直接作为中文阅读笔记的基础素材

## 📝 整理文稿示例结构

```
# 原文一级标题

正文内容（原文）
  📖 _中文翻译_

## 原文二级标题

正文内容（原文）
  📖 _中文翻译_

> *Caption: original caption text*
> 📖 *图注翻译文本*
```

> **提示**：翻译结果仅供辅助阅读参考，如需引用请对照原文核实。

## ⚙️ 配置与维护

### 在 `.env` 中配置翻译服务

```bash
# 阿里云翻译（推荐）
ALIYUN_TRANSLATE_ACCESS_KEY=你的AccessKey
ALIYUN_TRANSLATE_SECRET_KEY=你的SecretKey
ALIYUN_TRANSLATE_REGION=cn-hangzhou
```

### 获取阿里云翻译密钥

1. 登录 [阿里云控制台](https://www.aliyun.com/)
2. 搜索「机器翻译」→ 开通通用版翻译服务
3. 在 RAM 访问控制中创建 AccessKey
4. 填入 `.env` 文件对应字段

### 获取百度翻译密钥（备选）

1. 登录 [百度翻译开放平台](https://fanyi-api.baidu.com/)
2. 申请「通用翻译 API」服务
3. 获取 **AppID** 和 **密钥**
4. 填入 `.env` 的 `BAIDU_TRANSLATE_APP_ID` / `BAIDU_TRANSLATE_SECRET_KEY` 字段

> ⚠ 注意：百度翻译的凭证与百度 OCR 是两套独立的服务，需要分别申请。OCR 的 `BAIDU_OCR_API_KEY` 不能用于翻译。
