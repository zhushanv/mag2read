# Mag2Read SVG Icons

根据你给的 Mag2Read 设计稿整理的极简玻璃风 SVG 图标资源。

包含：
- 产品 Logo：书本 + 扫描线
- 上传云、PDF、图片、文档、下载
- 设置、亮暗模式、更多、展开
- 成功 / 处理中 / 失败状态
- AI 导读悬浮按钮、AI sparkle
- 空状态书本插画

大多数 SVG 使用 `currentColor`，可以用 CSS 直接改色：

```css
.icon-blue { color: #0071e3; }
.icon-success { color: #30d158; }
```

配套样式见 `mag2read-icons.css`，图标预览见 `preview.html`。
